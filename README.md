# console_minesweeper
Minesweeper for shell :s

## What is done

* [x] Base field with bombs and selection
* [x] Header printing
* [x] Color printing

## How to build and play

Run following in the project directory:

```bash
bash build.sh && ./build/console_minesweeper
```
